# Coloring book — Find the Province (v2)

Print-ready activity page matching the original layout with these upgrades:

- Numbered/lettered circles for coloring
- Three-step instructions plus a dotted example match
- Light interior detail on province outlines (lakes, foothills, peaks)
- Dotted regions on the background Canada map to color after matching
- Short clue labels under each icon
- Bonus box to draw another province symbol
- Footer page number **40**

## Files

| File | Use |
|------|-----|
| `find-the-province-game-v2.svg` | Edit in Illustrator, Inkscape, or Affinity; export PDF for print |
| `find-the-province-game-v2.png` | Preview at 2× letter size (1224×1584 px) |

## Print

- US Letter: open SVG, artboard **612×792 pt** (or 8.5×11 in), print at **100%**, single-sided.
- For KDP or professional print, export PDF with all text converted to outlines if your printer requires it.
